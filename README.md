## Giới thiệu
Website là đồ án môn lập trình web. It got 9đ. Phiên bản hiện tại là phiển bản không đầy đủ, hiện phiên bản đầy đủ đã bị mất.
Web siêu thị điện máy.

- Giỏ hàng
- Quản lý admin
- Đăng nhập,...
